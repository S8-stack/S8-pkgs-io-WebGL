Level of details

Level of details are defined below:

  • 3: Die hard, geared toward achieving the lowest possible performance footprint. hardly visible. Only acceptable for rendering of far field objects or during quick camera move.
  • 2: Same thing but less ugly.
  • 1: Ok for editing the geometry or day-to-day engineering.
  • 0: maximum detail for static high quality rendering.

Models

(WebGL_ObjectModel) object is the base description of an object, i.e. an engineered and somewhat independent representation of a real life object, like a pipeline, a pump, an exchanger, a probe, etc.

  • (WebGL_ShapeModel) feature[0]: code for feature 0. Fact is a shape object is a "piece" of an objectFields are:
    • (string) style: style to be applied. Special style "object" can be used to indicate that object style is to be used instead of model base style.
    • (WebGL_Mesh) lod[0]: mesh is level of detail of the shape. Can be of the following types WebGL_{Line, Triangle}Mesh. It includes:
      • pattern: of the shape within the mesh
      • WebGL_Vector{2,3}Buffers for vertices/normals/etc.
      • WebGL_{Segment,Triangle}Buffer
    • (WebGL_Mesh) lod[1]
    • (WebGL_Mesh) lod[2]
    • (WebGL_Mesh) lod[NB_LOD-1=3] (always 4 levels of details)
  • (WebGL_ShapeModel) feature[1]: code for feature 1
  • (WebGL_ShapeModel) feature[2]: code for feature 2
  • ...
  • (WebGL_ShapeModel) feature[i]: code for feature i

Instances

(WebGL_ObjectInstance) instance is the setup of a model within the context.

  • pattern of the object instance
  • (WebGL_ShapeInstance) feature[0] includes. Building block of rendering. It includes:
    • (rgb) picking color: picking color is defined on a per shape basis.
    • (boolean) isRendered: Enable/disable the rendering.
    • (WebGL_Renderable) lod[0]: renderable based on the level of detail of the corresponding mesh. renderable can be WebGL_{Line,Triangle}Renderable. It includes:
      • pattern: tensor product of the object instance pattern with the one of the corresponding mesh.
      • vertices attributes handles for the binding of the vertex buffers (vertices, normals)
      • handles for the triangles/lines buffer
    • (WebGL_Renderable) lod[1]
    • (WebGL_Renderable) lod[2]
    • (WebGL_Renderable) lod[3]
